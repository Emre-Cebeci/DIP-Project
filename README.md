### Installing and Running

1. Create a Python environment: <br>
   `python3 -m venv venv`

2. Activate the environment: <br>
   `source ./venv/bin/activate`

3. Install the required dependencies: <br>
   `pip install -r ./reqirements.txt`

4. Run: <br>
   `python3 ./main.py`
